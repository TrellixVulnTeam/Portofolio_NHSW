Sistem Kehadiran Berbasis IoT Dengan Face Recognition

Aplikasi Website Sistem Kehadiran yang digunakan di Kantor Desa Cimanggugirang.

Penjelasan Project

Project ini dibuat untuk keperluan absensi kehadiran di Kantor Desa Cimanggugirang agar tidak terjadi tindak kecurangan maka digunakanlah face recognition sebagai verifikasi pada saat melakukan kehadiran. Fitur yang dimiliki sistem ini adalah Data Pegawai, Data Absensi, Data Admin, dan Data Alat.
